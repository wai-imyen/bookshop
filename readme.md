# Yen's Second-Hand Bookshop ( 未完成 )

 網頁介紹：
>我的二手書拍賣網站，提供顧客二手書的選購。

開發動機：
>練習網頁製作。

開發應用：
>使用Laravel框架，以MVC架構開發，網頁符合RWD響應式設計，使用HTML、CSS、JavaScript、PHP、 Bootstrap等開發，資料皆從資料庫提取，網站功能含登入註冊、留言板、後台介面管理功能。
